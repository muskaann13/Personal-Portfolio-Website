Mini Amazon Clone

Amazon experience, re-imagined in mini.

Mini Amazon Clone

Mini Amazon Clone - a simple yet powerful e-commerce experience with API + localStorage. It is a demo e-commerce website built with HTML, CSS, and JavaScript that recreates the core shopping experience of Amazon. Product details are dynamically fetched from a backend API, making the site data-driven and closer to a real-world application. The project also implements localStorage to manage the shopping cart, ensuring that items remain saved even after refreshing or reopening the browser. With product listings, categories, and a responsive design, it demonstrates how modern e-commerce platforms handle data fetching, cart persistence, and clean UI design while giving users a simplified version of Amazon's shopping flow.
